<h1>Creando div's desde un objeto</h1>
<p>Tenemos el siguiente arreglo, el cual tiene a objetos como elementos. Cada objeto contiene la información de un trabajador de Laboratoria:</p>
<ol type="1">

     <li>Apellido</li>

</ol>
